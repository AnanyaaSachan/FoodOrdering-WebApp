# Food Ordering Web App

A modern food ordering web application built using React.js that allows users to browse restaurants, explore menus, manage cart items, and experience a smooth online food ordering interface.

## Features

* Browse restaurants and food items
* Dynamic restaurant menu pages
* Add to cart functionality
* Responsive user interface
* Category-based food listing
* Shimmer loading effects
* Search and filtering support
* Error handling pages
* Reusable component architecture
* Unit and integration testing setup

## Tech Stack
